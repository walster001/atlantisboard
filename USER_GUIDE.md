# User Guide

## Getting Started

### Registration and Login

1. **Register an Account**
   - Navigate to the registration page
   - Enter your email, username, and password
   - Password must be at least 12 characters with mixed case, numbers, and special characters
   - Click "Register" to create your account

2. **Login**
   - Enter your email/username and password
   - Or use Google OAuth if configured by your administrator
   - Click "Login" to access your account

### Creating Your First Workspace

1. Click "Create Workspace" from the dashboard
2. Enter a workspace name and description
3. Set visibility (Private, Team, or Public)
4. Click "Create" to finish

### Creating a Board

1. From your workspace, click "Create Board"
2. Enter board name and description
3. Choose a background color or image
4. Set board visibility and permissions
5. Click "Create" to finish

## Working with Boards

### Kanban View

- **Viewing Cards**: Cards are displayed in columns (lists)
- **Moving Cards**: Drag and drop cards between lists
- **Reordering Lists**: Drag list headers to reorder columns
- **Creating Cards**: Click "+" on a list to add a new card

### List View

- Access List view from the board view switcher
- Same functionality as Kanban, but with a list-based layout
- Cards are displayed in a vertical list format

### Board Settings

Access board settings by clicking the settings icon:
- **General**: Name, description, background
- **Labels**: Create and manage board labels
- **Members**: Add/remove members, manage permissions
- **Archive**: Archive or delete the board

## Working with Cards

### Creating Cards

1. Click "+" on any list
2. Enter card title (required)
3. Add description, labels, due date, etc.
4. Click "Create" to add the card

### Card Details

Click any card to open the detail view:
- **Title & Description**: Edit directly in the card
- **Labels**: Click to add/remove labels
- **Due Date**: Set due dates and reminders
- **Assignees**: Assign cards to team members
- **Checklists**: Create and manage task checklists
- **Comments**: Add comments and collaborate
- **Attachments**: Upload files and images
- **Activity**: View card activity history

### Card Actions

- **Duplicate**: Create a copy of the card in another list
- **Archive**: Remove card from view (can be restored)
- **Delete**: Permanently delete the card

## Advanced Features

### Labels

- **Creating Labels**: Only board admins can create labels
- **Assigning Labels**: Click label picker on any card
- **Color Coding**: Labels support custom colors
- **Filtering**: Filter cards by label

### Checklists

1. Open card detail view
2. Click "Add Checklist"
3. Enter checklist title
4. Add items to the checklist
5. Check off items as you complete them

### Comments

1. Open card detail view
2. Scroll to Comments section
3. Type your comment
4. Click "Save" to post
5. Edit or delete your own comments

### Due Dates and Reminders

1. Open card detail view
2. Click "Due Date"
3. Select a date
4. Set up to 3 reminders with custom timing
5. Reminders will notify you via in-app or push notifications

### Attachments

1. Open card detail view
2. Click "Attachments"
3. Select files to upload (up to 1000 MB per file)
4. Files are scanned for malware
5. Access attachments anytime from the card

## Templates

### Using Board Templates

1. Click "Create Board via Template"
2. Browse available templates
3. Select a template
4. Customize the template before applying
5. Create your board from the template

### Creating Card Templates

1. Open a card you want to use as a template
2. Click "Create Template" (if available)
3. Save with a name and description
4. Use the template when creating new cards

## Importing Data

### Trello Import

1. Export your Trello board as JSON
2. Go to board settings
3. Click "Import"
4. Select "Trello" format
5. Upload your JSON file
6. Monitor import progress
7. Access imported data when complete

### Exporting Data

1. Go to board settings
2. Click "Export"
3. Choose JSON or CSV format
4. Download the exported file

## Notifications

### Notification Types

- **Reminders**: Due date reminders
- **Assignments**: Assigned to a card
- **Comments**: Comments on your cards
- **Mentions**: @mentions in comments
- **Invites**: Workspace/board invitations

### Managing Notifications

1. Click the notification bell icon
2. View all notifications
3. Mark individual notifications as read
4. Mark all as read
5. Delete notifications you no longer need

### Notification Preferences

1. Go to your profile settings
2. Navigate to "Notification Preferences"
3. Configure preferences for each notification type
4. Choose delivery methods (in-app, push)
5. Save your preferences

## Permissions and Roles

### Role Types

- **Admin**: Full control, can manage members and settings
- **Manager**: Can manage boards, members, and settings (limited)
- **Member**: Can add/remove viewers, create and edit cards
- **Viewer**: Read-only access

### Managing Members

1. Open workspace or board settings
2. Go to "Members" section
3. Click "Add Member"
4. Enter email or username
5. Select role
6. Click "Add"

## Keyboard Shortcuts

- `n` - Create new card
- `f` - Focus search
- `?` - Show keyboard shortcuts
- `Esc` - Close modal/dialog

## Mobile Access

The application is fully responsive and works on mobile devices:
- Touch-optimized drag-and-drop
- Mobile-friendly card detail view
- Responsive navigation
- Install as PWA for app-like experience

## Getting Help

- Check the activity log for recent changes
- Contact your workspace admin for permission issues
- Review board/workspace settings for configuration options
- Check notification preferences if notifications aren't working
