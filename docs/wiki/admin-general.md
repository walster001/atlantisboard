# Admin: General Settings

The General Settings tab provides global application preferences that apply across all workspaces and boards.

---

## Accessing General Settings

1. Sign in as an **App Admin**.
2. Open the **Admin Configuration** panel from the navigation menu.
3. Select the **General** tab.

---

## Current Status

> **Note:** The General Settings tab is currently reserved for upcoming features. Future releases will add controls for:
>
> - Application name (global display name)
> - Default language
> - Default timezone
> - Session timeout settings
> - Other global preferences
>
> This page will be updated as new settings are added to the General tab.

---

## What's Available Now

While the General tab itself is pending expansion, many global settings are already configurable through other areas:

| Setting | Where to Configure |
|---------|--------------------|
| Authentication methods & registration | [Login Options](admin-login-options.md) |
| User roles & permissions | [Permissions & Roles](admin-permissions.md) |
| Login page appearance | Admin → Customisation → Login Branding |
| Application branding (navbar, fonts) | Admin → Customisation → App Branding |
| Email templates | Admin → Customisation → Email Branding |
| SMTP / email delivery | Admin → Email |
| Server configuration | [Environment Variables](environment-variables.md) |
