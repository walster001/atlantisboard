# Admin panel (compact)

[← Wiki home](Home.md)

Besides the large **Admin configuration** screen, there is a smaller **Admin panel** with a couple of tabs:

- **Permission roles** — quick access to role definitions.  
- **System configuration** — numbers that cap how aggressively the server throttles certain traffic (your admin adjusts these when the site is under load or attack).

If you are not an app admin, visiting these URLs sends you **home**.
