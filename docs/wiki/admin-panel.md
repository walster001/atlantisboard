# Admin panel (compact)

[← Wiki home](Home.md)

Besides the large **Admin configuration** screen, there is a smaller **Admin panel** with tabs such as:

- **Permission roles** — edit role definitions.  
- **System configuration** — a few numeric limits your administrator tunes for traffic and safety.

If you are not a site administrator, visiting these pages sends you back **home**.
