# Wiki screenshots

Add PNG (or WebP) files here and link them from wiki pages with a **relative** path from the page file.

**Examples**

- From `Home.md`, link: `images/home-overview.png`  
- From `boards-and-cards.md`, link: `../images/board.png` is wrong — keep all wiki pages next to `images/` so you can use `images/board.png`

**Naming**

Use short numeric or descriptive names: `01-login.png`, `home-empty.png`, `board-settings.png`.

**Privacy**

Do not commit screenshots that show real customer names, emails, or confidential board titles unless you have permission.
