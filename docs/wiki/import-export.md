# Bringing data in or out

[← Wiki home](Home.md)

---

## Import

From the **home hub**, open **Import** when you have permission. Supported sources depend on your server — common options include **Trello-style JSON** exports and spreadsheet-style files. Large imports may run as a **background job**; keep the tab open until a success message appears.

If import fails, read the error toast, fix the file, and try again. Your admin can check server logs for deeper causes.

---

## Export

Exports let you snapshot a board for backups or moving to another tool. Formats such as **JSON** and **CSV** are typical; your UI will list what is available.

Treat exports like sensitive company data — store them somewhere safe.
