# Import and export

[← Wiki home](Home.md)

---

## Import

From the **home** area, open **import** when you have permission. Available sources depend on your server (for example **Trello-style** exports). Large jobs may run in the background; wait for a success message before closing the tab.

If import fails, read the message, fix the file if needed, and try again. Your administrator can check server logs for technical causes.

---

## Export

**Export** saves a snapshot of a board (formats like **JSON** or **CSV** may be offered). Treat exports like confidential company data and store them safely.
