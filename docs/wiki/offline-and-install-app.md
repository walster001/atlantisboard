# Offline and install on your device

[← Wiki home](Home.md)

---

## Install shortcut

In supported browsers you may see **Install** or **Add to Home Screen**. That adds an icon that opens Atlantisboard like a small app.

---

## Offline

An **offline** notice can appear when the network drops. Some actions may **queue** and sync later depending on browser and server support. If the app says something did not save, treat that as final until you try again online.
