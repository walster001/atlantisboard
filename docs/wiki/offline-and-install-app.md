# Offline use and “install app” (PWA)

[← Wiki home](Home.md)

Atlantisboard can behave a bit like an **installable app** in supported browsers.

---

## Install / add to home screen

When the browser offers **Install** or **Add to Home Screen**, accept it. You get an icon that opens Atlantisboard in its own window without hunting for a bookmark.

---

## Offline

An **offline** notice may appear when you lose network. Some actions can be **queued** and synced later depending on your browser and server — if something did not save, the app will usually say so.

Do not rely on offline mode for critical work until you have tried it on your devices; behaviour varies by browser.
